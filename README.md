## 主な機能 (Feature List)
* **基本四則演算** (`+`, `-`, `*`, `/`)
* **連続計算のサポート**： `=`キーを押すことなく、連続した演算が可能です。
* **初期化機能**： `C` キーによる数値リセット、および `AC` (All Clear) 機能。
* **パーセンテージ計算**： `%` キーによる割合計算。
* **エラーハンドリング**： `0` での除算や数値オーバーフロー発生時のエラーメッセージ表示。
* **キーボード入力対応**： マウス操作だけでなく、テンキーおよび数字キーによる入力に対応しています。

## 使用技術 (Tech Stack)

    * **JavaScript** (ES6構文 / 計算ロジックの実装)
